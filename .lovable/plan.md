

# SnapKaza Landing Page

A premium, high-conversion landing page for an AI-powered luxury real estate marketing platform.

---

## Design System

**Colors**
- Background: Deep Charcoal (#121212)
- Accent Primary: Champagne Gold
- Accent Secondary: Electric Blue (for interactive elements)
- Text: White and muted grays

**Typography**
- Headlines: Playfair Display (elegant serif)
- Body: Inter (clean, modern sans-serif)

**Effects**
- Glassmorphism on navigation and cards
- Smooth scroll animations
- Hover effects on all interactive elements

---

## Technical Features

- **Mobile-first responsive design** (optimized for realtors on the go)
- **Smooth scroll animations** using Tailwind CSS
- **Hover effects** on all cards, buttons, and links
- **Intersection observer** for scroll-triggered animations
- **Accessibility** compliant with proper contrast and focus states
